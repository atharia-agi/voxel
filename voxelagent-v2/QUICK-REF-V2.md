# VOXELAGENT V2 - Quick Reference

**Philosophy:** Lightweight, Agentic-First, Addictive, Real Utility

---

## 🎯 One-Line Concept

> **Build your AI agent colony. Watch them work. Earn from their intelligence.**

---

## 🧊 Voxel as UI (NOT 3D World)

```
Your Workspace = Grid of voxel blocks
Each block = One AI Agent
No exploration, no navigation
Just CLICK + ASSIGN TASK + WATCH + EARN
```

---

## 🤖 8 Agent Types

| Type | Color | Function |
|------|-------|----------|
| Researcher | 🔵 Blue | Web research, data |
| Writer | 🟢 Green | Content, threads |
| Analyst | 🟡 Yellow | Market, trends |

| Social | 🔴 Red | Twitter, engagement |
| Trader | 🟠 Orange | Signals, arbitrage |
| Translator | ⚪ White | Multi-language |
| Designer | 🩷 Pink | UI, images |

---

## 💰 Token Flow

```
User pays $VOXEL → Spawn agent, assign task
Agent works (costs deducted)
Agent delivers output → User gets VALUE
5% tax → Treasury
Agent levels up → Better results
```

---

## 🔄 Agent Loop

```
1. SPAWN (pay $VOXEL)
2. ASSIGN TASK
3. WATCH AGENT WORK (live progress)
4. GET OUTPUT (research, content, etc.)
5. EARN XP + $VOXEL
6. LEVEL UP
7. UNLOCK MORE AGENTS
```

## 🛠️ Tech Stack (Lightweight)

| Layer | Tech | Why |
|-------|------|-----|
| Frontend | React + CSS 3D | No heavy WebGL |
| State | Zustand | Simple, fast |
| Backend | Next.js | All-in-one |
| Database | Supabase | Postgres + Vector |
| AI | LangChain + GPT-4 | Agent framework |
| Blockchain | Solana | Fast, cheap |

---

## 📊 Performance Target

- Initial Load: < 2s
- FPS: 60 (CSS animations)
- Memory: < 100MB
- Agent Response: < 30s
